# Desafio Técnico - Backend

## Rodando o Backend

Para rodá-lo, faça:

> cd BACK
> mv .env.sample .env
> yarn
> yarn start:dev

## Testando o Backend

Para rodá-lo, faça:

> yarn test:cov

Você também pode utilizar a collection do postman, path:

> BACK/postman
